# MANCOSA PGDBM Command Centre

Combined dashboard including:

- OB / ORB801 Organisational Behaviour
- IKM Information and Knowledge Management
- OM Operations Management
- MM Marketing Management

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

Framework: Vite  
Build command: `npm run build`  
Output directory: `dist`

`vercel.json` is included for React Router SPA rewrites.
